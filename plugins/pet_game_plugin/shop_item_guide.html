Guia de Criação de Itens para a Loja

1. Tipos de Itens Suportados

A loja suporta os seguintes tipos de itens, cada um com atributos específicos:

  • Chapéus (hat): Itens decorativos equipáveis na cabeça do mascote.
  • Acessórios (accessory): Itens como óculos ou luvas para o mascote.
  • Cor do Corpo (body_color): Altera a cor do corpo do mascote.
  • Formato do Corpo (body_shape): Modifica o formato do corpo do mascote (ex.: círculo, quadrado).
  • Consumíveis (consumable): Itens que fornecem efeitos temporários, como aumento de energia ou felicidade.

2. Estrutura de um Arquivo JSON

Os itens da loja são definidos por arquivos JSON armazenados na pasta shop_items. Cada arquivo representa um único item e deve seguir o formato abaixo:

Exemplo de Chapéu

{
    "id": "red_hat",
    "name": "Chapéu Vermelho",
    "type": "hat",
    "price": 50,
    "level": 1,
    "description": "Um chapéu vermelho estiloso."
}

Exemplo de Acessório

{
    "id": "cool_glasses",
    "name": "Óculos Descolados",
    "type": "accessory",
    "price": 75,
    "level": 2,
    "description": "Óculos escuros para deixar o mascote estiloso."
}

Exemplo de Cor do Corpo

{
    "id": "blue_color",
    "name": "Cor Azul",
    "type": "body_color",
    "price": 30,
    "level": 1,
    "color": "#0000FF",
    "description": "Deixa o corpo do mascote azul vibrante."
}

Exemplo de Formato do Corpo

{
    "id": "triangle_shape",
    "name": "Forma Triangular",
    "type": "body_shape",
    "price": 60,
    "level": 3,
    "shape": "triangle",
    "description": "Transforma o corpo do mascote em um triângulo."
}

Exemplo de Consumível

{
    "id": "energy_drink",
    "name": "Bebida Energética",
    "type": "consumable",
    "price": 20,
    "level": 1,
    "consumable_type": "energy_boost",
    "effect": { "amount": 20 },
    "description": "Aumenta a energia do mascote em 20 pontos."
}

3. Campos Obrigatórios

Certifique-se de incluir os seguintes campos em cada item:

  • id: Identificador único do item.
  • name: Nome do item exibido na loja.
  • type: Tipo do item (hat, accessory, body_color, etc.).
  • price: Preço do item em moedas.
  • level: Nível mínimo necessário para desbloquear o item.
  • description: Texto descritivo exibido na loja.

Campos adicionais para tipos específicos:

  • color: Código hexadecimal para body_color.
  • shape: Formato (ex.: circle, triangle) para body_shape.
  • effect: Objeto que define os efeitos para consumable.

4. Boas Práticas

  • Use IDs únicos para evitar conflitos entre itens.
  • Teste os novos itens antes de adicioná-los ao ambiente de produção.
  • Garanta que os arquivos JSON estejam bem formatados.
