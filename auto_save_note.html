Estoudo sobre Ponteiros no C++

Os ponteiros em C++ permitem manipular a mesmoria de modo dinamico então programador pode alocar e desalocar memoria manualmente assim podendo ter um codigo mais performance se souber oque está fazendo ou ter problemas de vazamento de memoria se não souber oque esta fazendo.

1. O que é um Ponteiro?

Um ponteiro é uma variável que armazena o endereço de memória de outra variável.
Declaração de Ponteiros:

int x = 10;
int* ptr = &x;  // ptr armazena o endereço de x

Aqui, ptr é um ponteiro para um int e guarda o endereço de x.

2. Ponteiros e Arrays

Em C++, o nome de um array é um ponteiro para seu primeiro elemento.

int arr[] = {1, 2, 3, 4, 5};
int* ptr = arr; // ptr aponta para arr[0]

std::cout << "Primeiro elemento: " << *ptr << std::endl;  // 1
std::cout << "Segundo elemento: " << *(ptr + 1) << std::endl; // 2

O deslocamento (ptr + 1) move o ponteiro para o próximo elemento.

3. Ponteiros e Alocação Dinâmic

Os ponteiros permitem alocar memória dinamicamente durante a execução usando `new e liberar com delete.

int* ptr = new int(10); // Aloca um int dinamicamente
std::cout << "Valor: " << *ptr << std::endl;
delete ptr; // Libera a memória

Para arrays:

int* arr = new int[5]; // Aloca um array de 5 inteiros
for (int i = 0; i < 5; i++)
    arr[i] = i * 10;

delete[] arr; // Libera o array
